# Contributing to Couri

Thanks for your interest in contributing! If you find a bug, have an idea for a feature, or want to improve the documentation, feel free to open an issue or submit a pull request. Keep changes focused, clearly described, and easy to review.

To contribute code, fork the repository, make your changes, and submit a pull request. Ensure your code follows the existing style and includes brief comments where necessary. Small, incremental improvements are welcome and every bit helps!
